# BUB Feedback

This repository exists for one purpose: **collecting bug reports and feature requests for the Bible Utility Belt (BUB) app.** There's no source code here — just the [issue tracker](https://github.com/evanda/bub-feedback/issues).

## Reporting a bug or requesting a feature

All feedback goes through GitHub Issues. You'll need a free GitHub account.

1. Go to the [**Issues** tab](https://github.com/evanda/bub-feedback/issues).
2. Search existing issues first — if your bug or idea is already there, add a 👍 or a comment instead of opening a duplicate.
3. Click **New issue**.
4. Write a clear, specific **title** (e.g. "Genealogy tree crashes when tapping Cain" rather than "bug").
5. Fill in the **description** and **Submit new issue**.

### What to include

A succinct description of the problem (or idea) and what you'd expect. Screenshots — with or without annotations — encouraged.

## About the app

**Bible Utility Belt (BUB)** is a Bible study app and your complete toolkit for studying Scripture. It is **offline-first** — all Bible text, commentary, lexicons, maps, and genealogy data are bundled on-device, so no network connection is needed at runtime. It collects zero data, is free with no ads, and uses only open-source or public-domain content. BUB runs as a web app in the browser and as a sideloadable Android APK (also on Google Play).

Six interconnected tools:

- **Bible Reader** — Berean Standard Bible (default) and KJV; tap any word for Strong's Hebrew/Greek definitions
- **Utility Belt** — a bottom sheet linking verses to concordance, original languages, era-filtered commentary from 200+ Church Fathers and Reformers, ~340,000 cross-references, and hymns
- **Atlas** — offline MapLibre map of the Holy Land with ~700 geocoded biblical places
- **Genealogy** — interactive family tree from Adam to Jesus with ~3,500 named individuals
- **Reading Plans** — date-based schedules with checkbox progress tracking saved locally

Learn more at [remingtons.org/bub](https://www.remingtons.org/bub).
